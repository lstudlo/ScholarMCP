## Prerequisites

- Node.js >= 20
- pnpm

## Commands

```bash
pnpm docs-old:dev
pnpm docs-old:check
pnpm docs-old:build
```

`docs:dev` and `docs:build` regenerate API/reference artifacts before running Astro.
