## Prerequisites

- Node.js `>=20`
- pnpm

## Setup

```bash
pnpm install
```

## Run from Source

Stdio mode:

```bash
pnpm dev:stdio
```

HTTP mode:

```bash
pnpm dev:http
```

## Verify

Type check package:

```bash
pnpm check
```

Run tests:

```bash
pnpm test
```
