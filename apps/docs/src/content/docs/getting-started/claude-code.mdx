This guide shows how to register ScholarMCP in Claude Code. Use the Claude CLI first, then fall back to manual config only if the CLI flow fails.

## What This Covers

- user-scoped setup across all your projects
- project-scoped fallback with `.mcp.json`
- `stdio` transport, which is the recommended mode for local MCP clients

Detailed setup steps are on this page.
