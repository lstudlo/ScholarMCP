This guide shows how to attach ScholarMCP to OpenCode using its MCP management commands and config file format.

## What This Covers

- interactive MCP registration with `opencode mcp add`
- verification with `opencode mcp list`
- manual fallback in `~/.config/opencode/opencode.json`

Detailed setup steps are on this page.
