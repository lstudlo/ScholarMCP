This guide shows how to register ScholarMCP in the Codex CLI and desktop app using the shared MCP configuration model.

## What This Covers

- CLI-first registration with `codex mcp add`
- verification with `codex mcp list` and `codex mcp get`
- manual fallback in `~/.codex/config.toml`

Detailed setup steps are on this page.
