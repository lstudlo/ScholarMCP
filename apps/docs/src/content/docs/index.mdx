ScholarMCP lets your coding agent run real literature workflows directly from chat:

- Search papers across Google Scholar, OpenAlex, Crossref, and Semantic Scholar
- Ingest and parse full-text PDFs from DOI, URLs, or local files
- Extract claims, methods, datasets, and limitations from parsed documents
- Build references and validate manuscript citations
